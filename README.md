# MathParser

A C++ port of [MathParser](https://github.com/SoftStoneDevelop/MathParser) project.<br>
Parsing and calculation of mathematical expressions.<br><br>
For example:<br>
Expression '2 + 3 * 4 - 5' - Result '9'<br>
Expression '(2 + 3) * 4 - 5' - result '15'<br>
